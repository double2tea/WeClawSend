WeClaw Send 使用说明

把文件拖到菜单栏纸飞机，直接发送到自己的微信。

macOS 14+Apple 芯片 + Intel无需 Xcode本地接口默认关闭
首次打开需要手动批准：当前版本没有 Apple Developer ID。请按下方第 2 步，在“系统设置 → 隐私与安全性”中点“仍要打开”。不要执行来源不明的终端命令。

三步开始使用

1

打开并安装

打开 DMG 或解压 ZIP，把 WeClaw Send 拖入“应用程序”。

2

批准首次打开

先打开一次，再到“系统设置 → 隐私与安全性”，点“仍要打开”。

3

扫码并发送

菜单栏点纸飞机 → 设置 → 扫码登录，然后拖入文件即可发送。

详细步骤

安装与首次打开

  1. 打开 DMG，或解压 ZIP。
  2. 把 App 拖入“应用程序”。
  3. 打开一次，看到拦截提示后关闭提示框。
  4. 打开“系统设置 → 隐私与安全性”。
  5. 找到 WeClaw Send，点“仍要打开”，再确认“打开”。

微信登录

  1. 点击菜单栏纸飞机。
  2. 打开设置，点“扫码登录”。
  3. 用手机微信扫码并确认。
  4. 若手机显示配对码，按提示输入。
  5. 看到“已登录”后返回发送页。

发送文件

  • 把文件直接拖到菜单栏纸飞机。
  • 或打开面板，拖到发送区域。
  • 也可以点击“选择文件”。
  • 最近传输会显示准备、加密、上传和完成状态。

设置与限制

  • 单文件最大约 200 MB。
  • 最多同时处理 3 个文件；加密和 CDN 上传可并行，微信消息提交保持串行且间隔约 2 秒。
  • 会话受限时 App 会提示你给 ClawBot 发一条消息，收到后自动重试原文件。
  • 本地接口默认关闭；仅脚本自动化和 Premiere 自动发送需要开启。
  • 换电脑后需要重新扫码登录。
  • 登录信息保存在当前用户的 Application Support，文件权限为 0600。

在线更新与编辑器安装

  • 设置中的“检查并更新”会从 GitHub Release 下载 App、校验 SHA-256，并在退出后替换和重启。
  • Premiere Pro 插件和 DaVinci Resolve Deliver 脚本均可在设置中一键下载、校验和安装。
  • App 更新要求所在目录可写；提示不可写时，请移至当前用户的“应用程序”目录。
  • 安装编辑器集成后，重启对应编辑器。

Premiere Pro 25+（可选）

  1. 在 App“设置 → 更新与编辑器集成”中点击 Premiere Pro 插件的“一键安装”。
  2. 重新打开 Premiere，在“窗口 → 扩展 → WeClaw Send”打开面板。
  3. 从下拉框选择自动扫描到的用户或系统预设。
  4. 需要导出后自动发送时，先在 App 设置中开启“本地接口”。

面板会扫描所有已安装的 Premiere 与 Adobe Media Encoder 25+ 预设，并记住上次选择；修改参数时，在 Premiere 原生导出页保存预设，再回面板刷新。项目只维护 CEP 12 版本，自动发送默认关闭。

DaVinci Resolve（可选）

  1. 在 App“设置 → 更新与编辑器集成”中点击 DaVinci Resolve 脚本的“一键安装”。
  2. 在 App 设置中开启“本地接口”。
  3. 重启 DaVinci Resolve，在 Deliver 页面使用“自动发送 ClawBot”脚本。

脚本只会安装到当前用户的 Deliver 脚本目录，不会修改项目文件或导出预设。

常见问题

打开后没有窗口

这是菜单栏应用，不会固定显示在 Dock。请查看屏幕右上角 Wi‑Fi、电池附近的纸飞机图标。

提示 App 已损坏

删除当前 App，重新打开原始 DMG 或解压原始 ZIP，然后拖入“应用程序”并按首次打开步骤批准。不要使用来源不明的终端命令。

发送失败

确认设置页显示微信已登录、网络可访问微信服务、文件没有超过 200 MB。若 App 提示刷新会话，请给 ClawBot 发任意消息，App 收到后会自动重试；必要时重新扫码。

脚本连接被拒绝

本地接口默认关闭。需要自动化时，在设置中打开“启用本地接口”，地址为 127.0.0.1:18790。
