01

安装

系统要求macOS 14 或更新 · Apple 芯片 / Intel 芯片
  1. 1

    下载并打开 WeClaw-Send.dmg；也可以下载 ZIP 后解压。

  2. 2

    把 WeClaw Send 拖进“应用程序”文件夹，然后双击打开。

  3. 3

    如果系统提示无法验证开发者，先关闭提示，再前往 系统设置 → 隐私与安全性。

  4. 4

    找到 WeClaw Send 被阻止的提示，点击仍要打开并确认。

  5. 5

    屏幕右上角出现纸飞机图标即安装完成。应用不会固定显示在程序坞。

为什么需要手动允许

当前发布包未使用 Apple Developer ID 签名，首次打开需由 Mac 所有者确认。

02

选择登录方式

有两种方式：独立扫码，或复用这台 Mac 已登录的 OpenClaw。

独立扫码

  1. 1

    点击菜单栏纸飞机，打开设置，再点击扫码登录。

  2. 2

    使用手机微信扫码并确认。如手机显示配对码，按提示填入并确认。

  3. 3

    在微信里给 ClawBot 发任意消息。应用取得可发送文件的会话后会显示“已连接”。

复用 OpenClaw

在“设置 → 登录方式”选择 OpenClaw。单账号会自动选中，多账号请手动选择。WeClaw Send 只读取当前账号和会话，发送时请保持 OpenClaw 运行。

扫码后没有连接

扫码确认不代表连接完成。先在微信里给 ClawBot 发一条消息。仍无反应时，取消后重新扫码，并暂时关闭代理或网络过滤工具。

03

发送文件

可以从这三处开始发送：

↗

拖到菜单栏

把 Finder 中的文件拖到右上角纸飞机。

＋

面板选择

点开纸飞机，选择一个或多个文件。

↓

拖到面板

把文件拖入面板中的虚线区域。

“最近传输”会显示准备、加密、上传和完成。失败项可以直接重新发送。

项目限制
单文件大小默认 200 MB，可选 100 MB、500 MB、1 GB 或 2 GB
并行处理最多 3 个文件
消息提交保持顺序，相邻两条至少间隔 2 秒
最近记录保留 20 条，只存在本机
04

常用设置

登录时自动启动

开机后自动出现在菜单栏，无需手动打开。

.mp4 显示为 .m4v

只改变微信附件的显示名，不修改 Mac 上的原文件。

单文件大小上限

调整应用本地校验。较大文件最终是否送达仍由微信服务决定。

本地接口

供 Premiere、DaVinci 和本机脚本调用，默认无需开启。

在线更新

发现新版本时会先显示更新内容。关闭后当前版本不再提醒。

05

剪辑软件接入

01

Premiere Pro

  1. WeClaw Send 设置 → Premiere Pro 插件 → 一键安装。
  2. 重启 Premiere，从“窗口 → 扩展 → WeClaw Send”打开。
  3. 选择导出预设和完整序列或 I/O 范围。
  4. 自动发送时，先在 App 设置中开启“本地接口”。

发送失败只重试发送，不重新渲染。

02

Final Cut Pro

  1. 打开“Final Cut Pro → 设置 → 目的位置”。
  2. 选择“导出文件”，把“操作”设为“其他”。
  3. 选择 /Applications/WeClaw Send.app。
  4. 使用该目的位置分享项目，完成后会自动发送。
03

DaVinci Resolve

  1. WeClaw Send 设置 → DaVinci Resolve 脚本 → 一键安装。
  2. 在设置中开启“本地接口”。
  3. 重启 Resolve，在 Deliver 页面使用自动发送脚本。

脚本只写入当前用户的 Deliver 脚本目录，不修改项目或导出预设。

06

故障排查

提示“无法验证开发者”

先尝试打开一次应用，然后到“系统设置 → 隐私与安全性”，找到拦截提示并点击“仍要打开”。

提示“已损坏，无法打开”

删除当前 App，重新打开原始 DMG 或重新解压 ZIP，再拖入“应用程序”。不要使用来源不明的终端命令绕过系统检查。

双击后没有窗口

这是菜单栏应用。查看 Wi‑Fi 和电池附近是否有纸飞机；菜单栏图标过多时，先展开隐藏项目。

能打开，但发不了

确认设置中显示“已连接”，检查文件是否超过上限。若使用 OpenClaw 登录，确认 OpenClaw 正在运行；若提示会话需要刷新，请给 ClawBot 发一条消息，应用取得新上下文后会自动重试。

发送一直排队

同时处理数已满 3 个时会排队；大文件的加密和上传也需要时间。请在“最近传输”查看当前阶段。

Final Cut 显示分享成功，但没有发送

确认已安装当前发布版，并重新选择 /Applications/WeClaw Send.app，避免目的位置仍指向旧安装。

07

隐私与安全

  • 不设开发者自有云端存储，文件通过微信通道发到你自己的 ClawBot。
  • 独立登录的凭据、偏好和传输记录保存在 ~/Library/Application Support/WeClawSend/。
  • 选择 OpenClaw 登录时，只读使用其本机状态目录中的当前账号与会话，不复制或修改 OpenClaw 文件，也不收取微信消息。
  • 可选本地接口默认关闭，只监听 127.0.0.1:18790，不会向局域网开放。
  • 普通菜单栏发送不依赖本地接口。
08

退出与卸载

退出：菜单栏纸飞机 → 设置 → 退出。

卸载：退出后，把“应用程序”中的 WeClaw Send 移到废纸篓。若要同时清除独立登录数据，再删除 ~/Library/Application Support/WeClawSend；不会影响 OpenClaw 的本机数据。
