剪辑软件

剪辑软件里直接发送

Premiere Pro、Final Cut Pro 和 DaVinci Resolve 都能接上。

Premiere Pro

导出完成后自动发送

选择完整序列或 I/O 范围，沿用 Adobe 导出预设。发送中可继续下一个序列。

  • 记住预设与输出目录
  • 失败只重试发送，不重新渲染
Final Cut Pro

原生分享动作，无需插件

把 WeClaw Send 设为导出完成动作，分享结束后自动进入发送队列。

  • 不需要开启本地接口
  • 可单独建立“导出并发送微信”目的位置
DaVinci Resolve

在 Deliver 页面直接发送

渲染完成后通过本机接口发送，可选视频或附件方式。

  • App 设置中一键安装
  • 脚本仅访问 127.0.0.1

登录方式

直接扫码，或复用 OpenClaw

独立扫码

扫码确认后，给 ClawBot 发一条消息完成会话绑定。

复用 OpenClaw

这台 Mac 已登录 OpenClaw 时，选择账号即可，无需再扫码。

只读使用

WeClaw Send 只读取登录信息和会话，OpenClaw 仍负责登录和收消息。

隐私

文件直接发给自己
登录数据留在本机

文件通过微信发给自己，不经过开发者服务器。独立登录的数据保存在本机；复用 OpenClaw 时，只读取当前账号和会话。

常见问题

安装和发送时
最常问的几件事

一定要安装 OpenClaw 吗+

不需要。WeClaw Send 可以直接扫码；已经在用 OpenClaw 时，选择复用就不用再扫码。

为什么扫码后还没显示“已连接”+

扫码确认后，再在微信里给 ClawBot 发任意消息。应用取得可发送文件的会话后，才会显示“已连接”。

复用 OpenClaw 时需要保持它运行吗+

需要。登录和收消息仍由 OpenClaw 管理，WeClaw Send 只读取当前账号和会话来发送文件。

首次打开被 macOS 拦截+

先尝试打开一次，再到“系统设置 → 隐私与安全性”选择“仍要打开”。

文件大小与并发限制+

默认单文件上限 200 MB，可在设置中调到 2 GB；最多同时处理 3 个文件。
