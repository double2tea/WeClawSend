开始之前

常见问题

一定要安装 OpenClaw 吗+

不需要。WeClaw Send 可直接扫码；已有 OpenClaw 时，才可选择只读复用它的微信登录和会话。

复用 OpenClaw 时需要保持它运行吗+

需要。登录与收消息仍由 OpenClaw 管理；WeClaw Send 只用当前账号和会话来发送文件。

首次打开被 macOS 拦截+

先尝试打开一次，再到“系统设置 → 隐私与安全性”选择“仍要打开”。

文件大小与并发限制+

默认单文件上限 200 MB，可在设置中调到 2 GB；最多同时处理 3 个文件。
